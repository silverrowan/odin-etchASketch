# etchASketch
Etch-a-sketch like project per The Odin Project 


Instructions
----------------
create webpage
in HTML: create div container for....
in JS: create grid of <div>s (16x16)
in CSS: use flexbox to make the divs appear as a grid

Set up a 'hover' effect - change grid div when mouse passes over them, leaving a trail.
Add a button at the top of the screen : send the user a pop-up asking for number of squares for a new grid
    once entered, the exisiting grid should be REMOVED, and a NEW grid generated in the *same space* (same # pixels).
    (set limit on user input to 100 max)

-----------------
bonus options:
    a. randomize the square's rgb color w ea interaction
    b. progressive darkening where each interaction w a square by 10% (fully black in 10 interactions)
